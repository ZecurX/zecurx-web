**SECURITY POLICY**

ZecurX Cybersecurity Pvt. Ltd. ("ZecurX") is committed to maintaining a secure, resilient, and compliant environment for all users, clients, and partners. This Security Policy describes the measures adopted to safeguard data, systems, and operational processes.

**1. Security Framework**

We follow industry-standard security practices aligned with ISO 27001, NIST, and other relevant cybersecurity frameworks.

**2. Technical Security Controls**

ZecurX maintains robust technical defenses, including:

-   End-to-end encryption of sensitive information

-   Firewalls, intrusion detection and prevention systems

-   Periodic vulnerability assessments and penetration testing

-   Secure cloud-based infrastructure

-   Continuous monitoring of networks and systems

**3. Access Management**

-   Role-based access control (RBAC) is enforced across all platforms.

-   Accounts require multi-factor authentication (MFA).

-   Access is granted strictly on a need-to-know basis.

**4. Employee Security Measures**

-   Employees undergo mandatory cybersecurity training.

-   Confidentiality and NDA agreements are required.

-   Permissions are reviewed and revoked upon role change or termination.

**5. Incident Response**

We maintain an incident response plan that includes:

-   24/7 monitoring and alerting

-   Immediate containment and remediation protocols

-   Root cause analysis

-   Detailed incident reporting to stakeholders

**6. Third-Party Security**

All third-party vendors handling data or infrastructure must comply with ZecurX security standards.

**7. Reporting Vulnerabilities**

Security concerns can be reported to: **security@zecurx.com**
